Navigation Bar Demo

This is a demo page showcasing the navigation bar component. The navigation bar includes:

  • Responsive design with mobile menu
  • Dropdown menus with different layouts (description, simple, icon)
  • Smooth animations and transitions
  • Accessibility features
  • Sign In and Get Started buttons

Try resizing your browser window to see the responsive behavior. On smaller screens, the navigation menu will collapse into a hamburger menu.
